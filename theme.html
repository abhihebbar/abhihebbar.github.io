Typography

Editorial serif for display. Highly readable sans for body and UI.

Hero / display
Thoughtful, clear, and deliberately understated.
Section heading

How headings should feel

Elegant, calm, high-contrast, never shouty.

Body copy

Body text should feel quiet and readable. The site should rely on typography and spacing rather than lots of decorative UI elements.

Type scale

  • Hero title: 56–72px desktop / 38–48px mobile
  • Section heading: 26–34px
  • Body text: 17–20px
  • Nav text: 14–15px
  • Eyebrow / label: 11–13px uppercase

Keep line lengths comfortable. Use fewer weights. Let size and spacing do the hierarchy work.

UI direction

Work Notes Contact
Engineer · Technical founder · Prototyper
Strong headline, short intro, calm call to action.

The page should feel structured by spacing and type rhythm, not by lots of boxes competing for attention.

Primary action Secondary link
Portrait should be secondary — author photo, not centerpiece.

Navigation

Small, quiet, minimal. No heavy nav background. No loud CTA button in the nav.

Buttons & links

Prefer links over buttons overall. One strong primary button style at most. No glossy gradients.

Cards & surfaces

Use sparingly. If present, they should feel like paper panels, not SaaS widgets or metric tiles.

UX rules

  • Immediate clarity over cleverness
  • Whitespace over unnecessary containers
  • One strong visual idea over many small tricks
  • Subtle motion only, if any
  • Responsive layouts should collapse gracefully, not turn into card soup
